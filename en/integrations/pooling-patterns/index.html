н’Җл§Ғ нҢЁн„ҙкіј мЈјмһ…

UNInjectлҠ” нҠ№м • н’Җ кө¬нҳ„мІҙлҘј к°•м ңн•ҳм§Җ м•ҠмҠөлӢҲлӢӨ.
лҢҖмӢ  MonoBehaviour мқёмҠӨн„ҙмҠӨк°Җ кәјлӮҙм§Ҳ л•ҢВ·лҸҢл ӨліҙлӮј л•Ң мЈјмһ… н•„л“ңлҘј л§һм¶”кі ,
мҪңл°ұмңјлЎң м „мІҳлҰ¬В·нӣ„мІҳлҰ¬лҘј кұё мҲҳ мһҲлҠ” л‘җ APIмҷҖ н•ң мқён„°нҺҳмқҙмҠӨл§Ң м ңкіөн•©лӢҲлӢӨ.

м ңкіө нҺ‘м…ҳвҖӢ

мҡ”мҶҢм—ӯн• 
IPoolInjectionTargetOnPoolGet / OnPoolRelease вҖ” мһ¬мЈјмһ… м§Ғнӣ„В·н•„л“ң мҙҲкё°нҷ” м§Ғм „
InjectTargetFromPoolTryInjectTarget(..., isReinjection: true) вҖ” м”¬В·м „м—ӯ н•„л“ң мһ¬н•ҙкІ°
ReleaseTargetToPoolOnPoolRelease нҳём¶ң нӣ„ [GlobalInject]В·[SceneInject] н•„л“ңлҘј nullлЎң

лЎңм»¬ [Inject] лІ мқҙнҒ¬ н•„л“ңлҠ” ReleaseTargetToPoolлЎң м§Җмҡ°м§Җ м•ҠмҠөлӢҲлӢӨ.

к¶ҢмһҘ нқҗлҰ„вҖӢ

1) мІ« мғқм„ұ(н”„лҰ¬нҢ№м—җм„ң мҠӨнҸ°)вҖӢ

ліҙнҶө SpawnInjected, InjectGameObject, лҳҗлҠ” InjectTarget мңјлЎң мІ« мЈјмһ…мқ„ лҒқк№ҒлӢҲлӢӨ.
мқҙл•ҢлҠ” isReinjection == false мқҙлҜҖлЎң, лҢҖмғҒмқҙ IInjected лҘј кө¬нҳ„н•ҳл©ҙ OnInjected() к°Җ нҳём¶ңлҗ©лӢҲлӢӨ.

// н’Җ мҙҲкё° мӣҢл°Қ: мІ« мқёмҠӨн„ҙмҠӨлҠ” мқјл°ҳ мЈјмһ… кІҪлЎң
var unit = subtreeInstaller.SpawnInjected(unitPrefab);
pool.Prewarm(unit);

2) н’Җм—җм„ң кәјлӮҙ м“ё л•ҢвҖӢ

мқёмҠӨн„ҙмҠӨлҘј мһ¬мӮ¬мҡ©н•ҳкё° м „м—җ ObjectInstaller.InjectTargetFromPool(instance) лҘј нҳём¶ңн•©лӢҲлӢӨ.
мЈјмһ…мқҙ м„ұкіөн•ҳл©ҙ IInjectedлҠ” нҳём¶ңлҗҳм§Җ м•Ҡкі , IPoolInjectionTarget.OnPoolGet л§Ң нҳём¶ңлҗ©лӢҲлӢӨ.

public class UnitPool
{
    public Unit Rent(ObjectInstaller scope)
    {
        var u = _stack.Count > 0 ? _stack.Pop() : scope.SpawnInjected(_prefab);
        scope.InjectTargetFromPool(u);
        u.gameObject.SetActive(true);
        return u;
    }

3) н’Җм—җ л°ҳнҷҳн•  л•ҢвҖӢ

ReleaseTargetToPoolмқ„ лЁјм Җ нҳём¶ңн•ң л’Ө, мҶҢмң  н’Җм—җ л„ЈмҠөлӢҲлӢӨ.

danger

мҲңм„ңлҘј л°”кҫёл©ҙ OnPoolRelease мқҙнӣ„м—җ м—¬м „нһҲ мӮҙм•„ мһҲлҠ” м°ёмЎ°лЎң лЎңм§Ғмқҙ лҸ„лҠ” мң„н—ҳмқҙ мһҲмҠөлӢҲлӢӨ.

    public void Return(ObjectInstaller scope, Unit u)
    {
        u.gameObject.SetActive(false);
        scope.ReleaseTargetToPool(u);
        _stack.Push(u);
    }
}
public partial class Unit : MonoBehaviour, IPoolInjectionTarget
{
    [SceneInject] private IBattleContext _ctx;

    public void OnPoolGet()
    {
        // _ctx л“ұмқҙ л§ү мЈјмһ…лҗЁ вҖ” кө¬лҸ…В·мғҒнғң лҰ¬м…Ӣ
    }

    public void OnPoolRelease()
    {
        // н•„л“ңк°Җ null лҗҳкё° м „ вҖ” мқҙлІӨнҠё н•ҙм ң
    }
}

ObjectInstallerмҷҖ л¶ҖлӘЁ мҠӨмҪ”н”„вҖӢ

_parentScope лҘј м§Җм •н•ҳл©ҙ мӢұкёҖнҶӨ м”¬В·м „м—ӯ мІҙмқё лҢҖмӢ  к·ё Installerмқҳ Resolve мІҙмқёмқ„ мғҒмң„лЎң м”ҒлӢҲлӢӨ.

н’Җ м „мҡ© м„ңлёҢнҠёлҰ¬(мҳҲ : мӣ”л“ң нҢЁнӮ· лӢЁмң„)лҘј кІ©лҰ¬н•  л•Ң мң мӮ¬н•©лӢҲлӢӨ.

danger
  • нӢұВ·IScopeDestroyableмқҖ Create<T> н•ңм •мқҙлҜҖлЎң н’Җлҗң MonoBehaviourмҷҖлҠ” лі„к°ңмһ…лӢҲлӢӨ.
  • н’Җ лЈЁнҠём—җ л¶ҷмқҖ ObjectInstallerк°Җ нҢҢкҙҙлҗҳл©ҙ, к·ё м•„лһҳм—җм„ң Create<T> лЎң л§Ңл“  м„ңл№„мҠӨмқҳ OnScopeDestroy к°Җ нҳём¶ңлҗ©лӢҲлӢӨ.
danger

мЈјмқҳн•  м җвҖӢ

  • мһ¬мЈјмһ… м—Ҷмқҙ мҳӨлһҳлҗң [SceneInject] м°ёмЎ°лЎң OnPoolGetл§Ң нҳём¶ңн•ҳлҠ” кІҪмҡ°,
    м”¬мқҙ л°”лҖҢм—ҲлҠ”лҚ° м°ёмЎ°к°Җ мқҙм „ м”¬ л§ӨлӢҲм ҖлҘј к°ҖлҰ¬нӮӨл©ҙ undefined лҸҷмһ‘мһ…лӢҲлӢӨ.
    м”¬ м „нҷҳ нӣ„ н’Җ мқёмҠӨн„ҙмҠӨлқјл©ҙ л°ҳл“ңмӢң InjectTargetFromPool(лҳҗлҠ” м „мІҙ нҠёлҰ¬ мһ¬мЈјмһ…) мқ„ кі л Өн•ҳм„ёмҡ”.

  • OnPoolReleaseм—җм„ң _ctxк°Җ мқҙлҜё null мқё мӢңм җмқҖ м•„лӢҷлӢҲлӢӨ. OnPoolRelease лҒқлӮң л’Ө SDKк°Җ н•„л“ңлҘј nullлЎң л§Ңл“ӯлӢҲлӢӨ.

м •лҰ¬вҖӢ

info

н’Җ кө¬нҳ„мІҙ(Stack, ObjectPool<T>, м„ңл“ңнҢҢнӢ°)лҠ” мһҗмң мқҙкі ,
UNInjectк°Җ кі м •н•ҳлҠ” кІғмқҖ InjectTargetFromPool / ReleaseTargetToPoolкіј IPoolInjectionTarget кі„м•Ҫлҝҗмһ…лӢҲлӢӨ.
мІ« мҠӨнҸ°кіј мһ¬мӮ¬мҡ©мқҳ мҪңл°ұмқ„ IInjected vs OnPoolGet мңјлЎң лӮҳлҲ„л©ҙ мқҳлҸ„к°Җ л¶„лӘ…н•ҙм§‘лӢҲлӢӨ.
